Installation
====================

This extension needs to be installed using composer. If you are not familiar with using composer
you can `read more about composer on the TYPO3 website <https://composer.typo3.org>`_.

Install the extension by running the following composer command:

.. code-block:: bash

	composer require typo3-ter/twig-for-typo3
